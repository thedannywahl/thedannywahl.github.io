New Moodle 2 theme: Discuss!

29 April 2011 on CSS, Git, Moodle, Theme

Well, I just “finished” my first Moodle 2.0 theme titled “Discuss!”.  The layout may or may not (but actually is) loosely based off of a similar sounding website.  I put finished in quotes above because this theme was my test pilot theme for learning the Moodle 2 theme engine and as such will likely have some errors, and I know it has too much CSS.  But hey, it might be a good inspiration point for someone else to work from.

You can grab it from my GitHub (yeah, I use git now, thanks to Tower) it’s called “Discuss4Moodle_2″ or you can grab it from the Moodle.org themes directory.  If you need support post in the Moodle.org thread for the theme.  Here’s a quick pic:
